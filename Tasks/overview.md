## Tasks Overview

This folder contains **high-level task lists** for each part of the project. Use these files as a guide for planning and assigning work – they are not implementation code.

### Files

- `client.md` – Tasks and summary for the dashboard UI (Tim).
- `routes.md` – Tasks and summary for routing and API contracts (Laurin).
- `backend.md` – Tasks and summary for backend services and storage (Jana).
- `cli.md` – Tasks and summary for the CLI command and its behavior (Tim + Jana).
- `non-functional-and-testing.md` – Cross-cutting work for performance, logging, stability, and testing.
